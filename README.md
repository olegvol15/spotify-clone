# LumiTune Frontend

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start local dev server
- `npm run lint` — run ESLint
- `npm run build` — type-check + production build
- `npm run preview` — preview production build locally
